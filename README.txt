Estudio Estadístico de la Variabilidad en los Precios de Productos Básicos

Este proyecto utiliza el lenguaje R para llevar a cabo un análisis estadístico que examina la variabilidad de los precios de productos esenciales, presentando los hallazgos mediante gráficos como histogramas, diagramas de barras y gráficos de dispersión.

Descripción General
El objetivo de este proyecto es investigar y comprender la dispersión de precios de diversos productos en distintas tiendas. A través de R, se calculan medidas estadísticas fundamentales como la media, mediana, moda, varianza y desviación estándar. Asimismo, se crean visualizaciones para representar gráficamente los resultados, facilitando la detección de patrones y anomalías.
Principales Características

Cálculo de medidas de tendencia central: media, mediana, y moda.
Cálculo de medidas de dispersión: varianza y desviación estándar.
Visualización de datos mediante gráficos: histogramas, diagramas de barras y gráficos de dispersión.

Requisitos Previos
Asegúrate de que los siguientes paquetes estén instalados en R antes de ejecutar el código:
install.packages("readxl")  
install.packages("plotly")  
install.packages("modeest")  
install.packages("dplyr")  
install.packages("qcc")  

Instalación
Clona el repositorio utilizando el siguiente comando:
git clone https://github.com/"TuNombreDeUsuario"/"TuRepositorio".git  


Instrucciones de Uso
Carga los paquetes requeridos utilizando el script load_packages.R. Luego, ejecuta el script principal analisis_precios.R para cargar los datos y generar las visualizaciones.
